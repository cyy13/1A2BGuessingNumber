# 1A2BGuessingNumber
Online guessing number game in Ethereum.

## Getting Start
```
npm install
npm start
```

### Deploy Github Pages
```
npm run deploy
```

### Compile Contracts

```bash
npm run compile 
 ```
If it's not working on windows, try to install a global truffle

```bash
npm i -g truffle
truffle compile && cp -r build/contracts src/
```
